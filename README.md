# How to run example project

## setup sybolic link for runtime development
```
  $> npm install
  $> ln -fs ~/PATH_TO_THIS_PROJECT/src/ ./example/src

  it should be like this:

  --- editor_project_home
   |--- example
   |  |- src
   |--- src
   |---  

```

## run npm script
```
  $> npm run start
```
